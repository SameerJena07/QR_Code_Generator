# 📱 QR Code Generator

A simple and responsive **QR Code Generator Web Application** built using **HTML, CSS, and JavaScript**.
This application allows users to enter any text or URL and instantly generate a **QR code** that can be scanned using a smartphone.

---

## 🚀 Features

* Generate QR codes instantly
* Convert **text or URL into QR code**
* Simple and clean user interface
* Responsive design (works on mobile and desktop)
* Fast QR code generation
* Easy to use

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout design
* **JavaScript (Vanilla JS)** – Logic for generating QR codes

## 💡 How It Works

1. The user enters **text or a URL** in the input box.
2. The user clicks the **Generate QR Code** button.
3. JavaScript sends the input to a **QR code API/library**.
4. The QR code image is generated and displayed on the screen.

---

## 📸 Example

Input:

```
https://google.com
```

Output:

A QR code will be generated that redirects to **Google** when scanned.
